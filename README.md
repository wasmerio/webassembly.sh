# WebAssembly.sh

Open-source and installable Progressive Web App (PWA) terminal fully powered by WebAssembly and Wasmer-JS, that uses the [WebAssembly Package Manager (WAPM)](https://wapm.io/) and local files to run server-side Wasm WASI modules in a shell-like interface  üñ•
